Built a forecasting system for Kaohsiung, Tainan, and Pingtung using Random Forest, XGBoost, LSTM, and a hybrid LSTM–Transformer. 1. Project Overview

Motivation

Dengue outbreaks in southern Taiwan are driven by climate and mosquito dynamics, but early-warning tools are limited. This project builds a forecasting system for Kaohsiung, Tainan, and Pingtung using multi-source data to support timely public health action.

Objectives

  • Construct a unified weekly dengue–climate–mosquito dataset for southern Taiwan.
  • Compare four forecasting models: Random Forest, XGBoost, LSTM, LSTM–Transformer.
  • Identify key climatic and entomological drivers of dengue risk.
  • Deploy interactive dashboards using ArcGIS, Streamlit, and Supabase for decision support.

2. Methodology & Models

Data Pipeline

  • Sources: Taiwan CDC (dengue & BI/HI/CI), NCHC weather & rainfall.
  • Temporal Scale: Aggregated to weekly city-level time series.
  • Preprocessing: Outlier filtering, interpolation, forward/backward fill, lag feature engineering.
  • Scaling: Min–Max (deep learning), z-score (tree models).

Evaluation Setup

  • Time-based train/validation/test split (train up to 2017, validation 2018–2021, test 2022–2024).
  • Metrics: RMSE, MAE, MSE, and R² on held-out weekly test data.

Models Compared

Random Forest

Ensemble of decision trees with time-based cross-validation and hyperparameter tuning.

XGBoost

Gradient-boosted trees with grid search over depth, learning rate, and tree count.

LSTM (Lag Features)

Sequence model on standardized lagged features with sample weighting during outbreaks.

LSTM–Transformer

LSTM encoder + multi-head self-attention and cyclic calendar features for seasonality.

3. Key Results – Model Comparison

Across all three cities, tree-based models (Random Forest and XGBoost) consistently outperformed LSTM-based models in predicting weekly dengue cases, especially during large outbreaks.

Kaohsiung City

Model RMSE MAE R²
Random Forest 19.28 6.90 0.90
XGBoost 14.92 5.24 0.95
LSTM (Lag) ≈19–22 ≈7–9 0.80–0.88
LSTM–Transformer ≈18–21 ≈7–8 0.82–0.89

Best model: XGBoost (lowest RMSE, highest R²).

Tainan City

Models trained independently with similar feature sets and evaluation design.

  • XGBoost and Random Forest again outperformed LSTM models.
  • LSTM variants captured general seasonal trends but underpredicted peak outbreaks.

Pingtung County

Smaller case counts and fewer large outbreaks produced slightly noisier signals.

  • Tree-based models remained more robust to sparse and irregular peaks.
  • LSTM models occasionally smoothed away short, sharp outbreaks.

4. Interactive Tools

4.1 ArcGIS Online Dashboard

An ArcGIS Online dashboard was developed to visualize spatiotemporal dengue patterns, mosquito indices, and rainfall intensity across Taiwan from 2010–2025. This dashboard highlights the concentration of outbreaks in Southern Taiwan and supports exploratory analysis of space–time clustering.

4.2 Streamlit Forecasting App

A Streamlit web application allows users to explore EDA, ACF/PACF, model performance, and forecast diagnostics for each city. Users can switch between Random Forest, XGBoost, LSTM, and LSTM–Transformer to compare test metrics and view predicted vs. actual time series.

5. SWOT Analysis

The table below summarizes the strengths, weaknesses, opportunities, and threats of the proposed dengue forecasting framework and its implementation in Southern Taiwan.

Strengths

  • High predictive accuracy of Random Forest and XGBoost across cities.
  • 15-year multi-source dataset integrating dengue, climate, and mosquito indices.
  • Interpretable feature importance supports public health decision-making.
  • Interactive dashboards (ArcGIS + Streamlit) for real-time exploration.

Weaknesses

  • Weekly temporal resolution limits deep learning performance.
  • LSTM-based models underpredict large outbreak peaks.
  • Irregular mosquito index data require imputation.
  • City-level granularity; lacks neighborhood-level spatial precision.

Opportunities

  • Incorporate mobility, search trends, and satellite imagery.
  • Extend to advanced spatiotemporal models (TFT, GNN, Seq2Seq).
  • Build automated early-warning pipelines for health authorities.
  • Scale to district or village-level risk maps for targeted control.

Threats

  • Climate change may alter dengue seasonality and intensity.
  • Surveillance changes, reporting delays, or inconsistent data.
  • Emerging serotypes or vectors may reduce model stability.
  • Rapid human mobility can trigger unexpected outbreak clusters.
